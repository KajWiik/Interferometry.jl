hub‐ci‐status(1)  ‐‐ Display GitHub Status information for a com‐
mit.  ===

## SYNOPSIS

‘hub ci‐status‘ [‐v] [<COMMIT>]


## OPTIONS
  * ‐v:
    Print detailed report of all status checks and their URLs.

  * <COMMIT>:
    A commit SHA or branch name (default: "HEAD").

Exits with one of: success (0), error (1), failure  (1),  pending
(2), no status (3).

## SEE ALSO

hub‐pull‐request(1), hub(1)
